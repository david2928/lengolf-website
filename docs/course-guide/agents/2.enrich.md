# Stage 2: Enrich

**Who does this:** Claude — runs automatically after Stage 1

## What happens

Claude reads each raw JSON file from `data/1.raw/<region>/` and adds prose sections.
The structured fields are never changed at this stage — only prose is added.

For each course, Claude writes:

1. **overview** — 150-200 words. Course reputation, experience, what makes it distinctive. Factual, not marketing copy.
2. **layout_and_experience** — 250-350 words. Landscape, terrain, difficulty, notable holes, course style. No fabricated specifics — only what the data supports or what is well-documented.
3. **tips** — 150-200 words. Best tee times, pace of play, what to bring, weather considerations for this specific location.
4. **location_and_access** — 100-150 words. Distance from Bangkok, typical drive time, transport options (taxi, Grab, Golf Link shuttle if applicable). Drawn directly from `distance_from_bangkok_km` and `drive_time_from_bangkok_min` fields.
5. **rental_cta_context** — 1-2 sentences specific to this course. Used in the introductory CTA block. Example: "Playing Nikanti Golf Club during your Bangkok trip?" Not a generic line.

Total target: 800-1,200 words of prose per course for Phase 1 (top 30).

## Rules for prose writing

- No invented specifics. If Claude doesn't have a verified fact for a claim, say "approximately" or leave it out.
- No superlatives without evidence ("one of Thailand's best" requires a ranking source in the data).
- All distances and drive times must come from the `fields` object — not estimated from Claude's knowledge.
- The rental CTA must reference the specific course by name, not use a generic template.

## Output format

`data/2.enriched/<region>/<slug>.json`

Same structure as Stage 1, with a `prose` object added:

```json
{
  "slug": "nikanti-golf-club",
  "region": "bangkok",
  "status": "enriched",
  "collected_at": "2026-04-16",
  "enriched_at": "2026-04-16",
  "fields": { ... },
  "prose": {
    "overview": "Nikanti Golf Club is a...",
    "layout_and_experience": "Set in Nakhon Pathom province...",
    "tips": "Tee times before 7am are recommended...",
    "location_and_access": "Nikanti Golf Club is located approximately 35km west of Bangkok...",
    "rental_cta_context": "Planning to play Nikanti Golf Club during your Bangkok trip?"
  }
}
```
