- Green fee: 3,650 THB all-inclusive (green fee + caddie + cart) — foreigner flat rate, every day
- Note: Thai/ID-holding foreigner rate: 1,150 THB weekday / 1,750 THB weekend (all-in)
- Caddie fee: null (bundled) | Cart fee: null (bundled)

- Distance from Bangkok: ~130 km / ~100 min
- GPS: 13.023781, 101.424955

2. Address inconsistency: Sources list both "137 Moo 1" and "129 Moo 6" at the same Pa Yup Nai, Wang Chan location. Could not resolve definitively without direct contact.
3. Jungle Course status: Intermittently closed for maintenance per TripAdvisor reviews. Some sources count 36 holes, others 54. The three-course figure (54 holes) is the facility's stated capacity.
