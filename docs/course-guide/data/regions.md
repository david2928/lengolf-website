# Master Course List — Thailand

All known golf courses in Thailand, organised by region.
Sources: All Square Golf (214 courses), Leading Courses (163), Hole19 (242), GolfAsian, Golf Holidays Thailand, GolfSavers, GolfLux, GolfPass.
Compiled: 2026-04-16

Phase 1 courses (priority content, tourist-facing, 18-hole) are marked ★.
9-hole courses are marked (9H).
Military/air force/army courses are marked (MIL).
Dam/government courses are marked (GOV).
